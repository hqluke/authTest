# Workout Tracker

A full-stack workout and running tracker built with Node.js, Express, 
and EJS, backed by PostgreSQL.

## Demo
https://drunk-billy-hqluke-34090c79.koyeb.app/demo

## Features
- Log and track workouts and runs
- Calendar view of activity history
- User authentication and accounts
- Demo mode to try without signing up

## Tech Stack
- Node.js / Express
- EJS (server-side rendering)
- PostgreSQL
- Passport.js, Bcrypt, Helmet, express-rate-limit
